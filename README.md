# mcs_beacon

Eddystone + iBeacon Linux implementation

## Bleno

run like that:

### Eddystone
```
sudo node eddystoneBeacon.js
```
### iBeacon
```
sudo node iBeaconBeacon.js
```
### Both (Main program)
```
sudo node beacoNeando.js
```
Prerequisites:

- Use node 8.9.4
- See Bleno prerequisites (https://github.com/noble/bleno)
